# Calendar Features

## Schedule Features
### Academic Year
The user inputs the stard date and end date of the school year, the values are stored in the user configuration along with the academic year (e.g. 2025-2026). 
### Holiday
The user inputs the holiday dates or periods and the list of dates is stored in the configuration. Based on the sorting of the diary topics, the program adjusts the expected dates when the topic should be taught to each class.
### Swap Days
Swap days are days when the user has the schedule of a different day of the week (e.g. Monday schedule on a Wednesday). This changes the expected date of when a topic should be taught to a class. The user inputs the swap days and the list of the special dates with a different days program, along with the special day of the week that will be replacing the actual day of the week is stored in the configuration (e.g. 2025-11-28, Monday). Based on the sorting of the diary topics, the program adjusts the expected dates when the topic should be taught to each class.
