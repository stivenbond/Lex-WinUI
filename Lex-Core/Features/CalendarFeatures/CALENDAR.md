# Calendar Features

## Schedules and deadlines.
    - The user can set the start and end date of the academic year. This is stored in the configuration. and can be changed later on.
    - The user can set dates or periods ( via start and end dates ) that are considered holidays where school is not in session.
    - The user can set dates where schedule waps occur. On a given date, the schedule of a day of the week is going to be taught. These swap dates can be any of the 7 days of the week.
    - The user can set dates along with periods where the teachers take over periods ( kur nje mesues ka me shume ore dhe jep, ose merrem ore te lira)

## Syllabus Completion pace
    - The syllabus' topics are organised sequentially along with the progress of the available periods where school will be in session and the teach is toing to teach their subject. The pace of the topics taught is based on the availaple periods.
