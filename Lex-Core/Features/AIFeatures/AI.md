# AI-Related Features

## Unprompted Generation
    - The user has access to an AI general-purpouse model, that can be given context from different parts of the app and then answer questinos based on that information, or generate content based on the context.

## Prepared Structured Generation
    - The user can generate Diary Content, based on the schema they use, with AI, using the context from the specific lesson topics. 
    - The user can use AI to generate 30 tests of different variants, using the context from the specific Lesson topic. The AI can generate questions of different types, and then the questions will be concatenated randomly to compile different tests.
    - The user can use AI to customize an existing, comprehensive test, to create 30 different sub-versions of the test. [ Questions can be rephrased, Choices shuffled or changed, Question types can be changed, ]
    - The user can use AI to generate diagrams, based on the context from the specific lesson topic.
    - The user can use AI to generate flashcards, based on the context from the specific lesson topic.
    - The user can use AI to generate  outines of each lesson topic, and reorganize the written content of the lesson to be more precise and structured.
    - The user can use AI to generate concise presentations based on the context from the specific lesson topic.
