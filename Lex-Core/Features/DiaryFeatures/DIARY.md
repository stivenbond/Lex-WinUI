# Diary Features

## Diary Entry Features

### Diary Entry Base Schema
- the base schema and the actual schema are the same, but the actual schema can be modified by the user
- the base schema is stored in the configuration as base_schema
- the actual schema is stored in the configuration as actual_schema
- the user can reset the actual schema to the base schema
- the user can create a new diary entry with the actual schema by default but has the option to use the default schema any time
- the user can add notes to any referenced diary entry nd can choose on a per-category basis if the principal entry will be referenced or if the content will change or if the whole entry will be referenced and the user will make annotations below the referencing statement.
